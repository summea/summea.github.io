  • Mailing List Sync (2018-Present)

    Scripts that sync mailing lists between Google Groups and Mailman. This project is a collection of individual Perl scripts that:

    1. Get a list of all available organization mailing lists from Ume API
    2. Get all members of each individual list using Google's API
    3. Save found members to individual lists for Mailman
    4. Import mailing list changes into Mailman
    5. Update sending rights for users with mailing list changes

    Previously, this project ran the first three steps all at one time, but this led to times where Ume API was so busy with the related requests that it couldn't handle new incoming requests. Thus, there would be times where Ume API was unavailable during some of the processing taking place with the Mailing List Sync. In 2020, there was the idea to address this unavailability issue by using the task queue program called Celery. So now, when requests for mailing list members are made, they are taken up as tasks to run independently, so Ume API now can fire off tasks and provides a result URL that can be checked for when the task is completed. Once completed, the desired information is returned without causing a slowdown for Ume API, which means that it can continue to handle other incoming requests as expected.

    Technology Used
    celery mailman perl ume-api
